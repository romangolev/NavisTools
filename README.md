# NavisTools

Productivity tools for Autodesk Navisworks.

## Features

- **Add Parent Name** - Add parent element names to selected items as parameters
- **Total Sums** - Calculate total sums for selected items
- **Selection Info** - View detailed information about selected elements

## Installation

Download and run the latest installer from [Releases](https://github.com/romangolev/NavisTools/releases).

## Build

```bash
nuke CreateInstaller
```
